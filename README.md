# Password Generator 🔐

A simple command-line password generator written in C++.

## What does it do?

Generates a random password with letters, numbers, and special characters. You choose the length!

## How to run

1. Open the project in CLion
2. Press the Play button ▶️
3. Enter the desired password length

## Example

```
Enter the length of the password: 12
aB3#kLm9!Xqz
```

## Features

- Lowercase & uppercase letters
- Numbers
- Special characters (!@#$%)
- Custom password length

## Built with

- C++20
- CLion
